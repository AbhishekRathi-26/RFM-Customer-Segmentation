# RFM Customer Segmentation Analysis

## Project Overview

This project performs **RFM (Recency, Frequency, Monetary) Analysis** to segment retail customers according to their purchasing behaviour.

The objective is to identify high-value customers, inactive customers, loyal customers and customers with future growth potential using Python (Pandas) and Tableau.

## Dataset

Online Retail Transaction Dataset

---

## Business Problem

Retail businesses often struggle to identify:

- High-value customers
- Loyal customers
- Customers likely to churn
- Customers suitable for marketing campaigns

This project solves this problem using RFM Analysis.

---

## RFM Metrics

### Recency

Days since customer's last purchase.

### Frequency

Number of purchases made.

### Monetary

Total amount spent.

## Business Insights

- Champions generate the highest revenue.
- Lost customers show the highest recency.
- Loyal customers purchase frequently.
- Potential Loyalists can be converted using marketing campaigns.
- Others represent the largest customer base.

---

## Business Recommendations

- Reward Champion customers.
- Launch loyalty programs.
- Re-engage Lost Customers.
- Convert Potential Loyalists.
- Target Others through promotions.
